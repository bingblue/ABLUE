# agroup
群官网，Android版，基于martial design

**群名称** ： JAVA、Android、WEB、前端

**群号** ： 206683621

## 项目功能 (/agroup)

- 首页(/index)
- 登录(/login)（南京-zing）
- 登出(/logout)（南京-zing）
- 注册(/reg)
- title
- 底部导航
- 左侧栏

*Tips:* 功能参考各大APP，如QQ、网易等，UI基于martial design库，**icon使用fontawesome**。

## 合作规范

- 服务器端API请移步根目录下的[API.md](https://github.com/bingblue/agroup/blob/master/API.md)
